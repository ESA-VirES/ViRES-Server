Changelog

Release of Service v1.1 (26.09.2016)

Features:

  • Label with colorscale information directly integrated into globe visualization
    Now when using the save function the currently configured color scale and ranges are saved also into the image. It is possible to toggle the visualization of the Label in the settings panel for each Layer.
  • Dynamic range calculation for models
    When changing model parameters (SIFM, IGRF12, CHAOS-5, Custom Model), such as the coefficient range or height the range (or domain) for the colorscale is calculated and set automatically. Of course it is still possible to manually adjust the range with both textfields available in the settings panel. This change was introduced as it became clear that finding the correct range after changing model parameters was cumbersome for users and we wanted to facilitate the process.
  • Single axis interaction
    When exploring the scatter plot, one can scroll when hovering over an axis to zoom along one axis while keeping the other axis range fixed. Similarily one can drag along an axis to pan the plot. These features apply to both the x-axis and the y-axes and complement the zooming and panning one can perform on the scatterplot itself to zoom along both axes.
  • Zoom limit removed
    The zoom limit previously applied to the scatterplot is now removed. There are now no restrictions to how far you can zoom in or out. A reset button is added to enable the user to return to the initial zoom level and position at one's convenience.
  • Customizable filter list
    The data filter histograms found in the Analytics View can be removed individually, and new histograms for any other available parameter can be added by selecting it in the histogram "Add" dropdown menu. Any active filters will be removed when its corresponding histogram is removed. In addition the filters can be hidden to allow for more space for the scatterplot.
  • Secondary y-axis
    When selecting multiple parameters a second y-axis will appear. When selecting two parameters this means that each parameter has its own axis. If more than two parameters are selected, they will be grouped according to their range, such that the parameters with the most similar value ranges are grouped together and share an axis.
  • Scatterplot units
    The units are now displayed in a paranthesis next to the axis parameter labels in the scatterplot.
